Codex + Claude plugin package

Chaos Substrate

Install the plugin once, bootstrap the local services once, then let your agent index, query, and generate feature-memory websites from any Rust, TypeScript, or JavaScript project.

One-Time Setup

CHAOS_CONFIG=chaos-substrate.local.toml scripts/chaos-agent bootstrap
export PATH="$HOME/.local/bin:$PATH"

Use this from the Chaos Substrate checkout. It installs the wrapper, starts Ollama when configured, builds the binary, starts Postgres, runs migrations, and checks the system.

Project-Level Use

Use Chaos Substrate on this project and create an index plus explanation.
Generate a feature explanation website for authorization and RBAC.
Find implementation context for authorization and RBAC.

Ask the agent naturally after the plugin is enabled. The plugin skill chooses the wrapper command; direct CLI use is for debugging.
